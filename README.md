# ✈️ AeroVista Flight Booking System

A full-stack Flight Booking System built using the MERN Stack.

## Features

### User Features

* User Registration & Login
* JWT Authentication
* Browse Available Flights
* Search Flights
* Seat Selection
* Flight Booking
* Cancel Booking
* Profile Page
* My Bookings Page

### Admin Features

* Admin Dashboard
* Add Flight
* Edit Flight
* Delete Flight
* Manage Flights
* View All Bookings
* View All Users

## Tech Stack

### Frontend

* React.js
* React Router
* Tailwind CSS
* Axios
* React Hot Toast

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* bcryptjs

## Project Structure

flight-management-system

* client
* server

## Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the server folder:

```env
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_secret_key
PORT=5000
```
